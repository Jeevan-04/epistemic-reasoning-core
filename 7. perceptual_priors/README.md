# Perceptual Priors

**NON-INFERABLE, NON-INHERITABLE perceptual knowledge**

## Purpose

Models knowledge from direct observation, not logical derivation.

## Design Principles

1. **Non-inferable**: Cannot be derived through reasoning
2. **Non-inheritable**: Do not propagate taxonomically
3. **Lower confidence**: Always < logical beliefs (max 85%)
4. **Explicitly labeled**: Marked as "perceptual" not "logical"

## Examples

- "Gold is shiny" - because we saw it, not because logic proved it
- "Water is liquid" - observational fact at room temperature
- "Copper is conductive" - empirical property

## Not Perception

This is NOT a perception system. It's a module for perceptual PRIORS - learned associations that mirror how humans store observational knowledge separately from logical inference.

## Academic Framing

> "Certain properties (color, texture, conductivity) are modeled as perceptual priors, not logical conclusions. This reflects the epistemic distinction between observation and inference."
